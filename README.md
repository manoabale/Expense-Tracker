# Expense-Tracker
A lightweight app to track daily expenses and calculate total spending.

🚀 Features

• Add new expenses
• Auto‑calculate total
• Simple list display
• Minimal UI


🧩 Tech Stack

• HTML
• CSS
• JavaScript


📂 Project Structure

index.html
style.css
script.js


📸 Preview

Shows a form for adding expenses and a list of all entries.

🧠 How It Works

Each expense is stored in an array and rendered into a list. The total is recalculated every time a new expense is added.
